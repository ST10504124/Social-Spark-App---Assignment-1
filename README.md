# Social-Spark-App---Assignment-1

Social Spark App
Description
Social Spark is an Android application built using Kotlin in Android Studio. The app provides social interaction suggestions based on the user's input for the time of day.
By entering a time (e.g., Morning, Afternoon, Dinner), the app generates a relevant "social spark" to encourage meaningful connections with others.

Objective

The goal of this project is to:
•	Build an interactive Android app using Kotlin
•	Implement conditional logic (if statements) to generate suggestions
•	Practice version control using GitHub
•	Integrate GitHub Actions for automated builds

Features

•	User input for time of day
•	Social suggestions based on input
•	Reset button to clear input and results and a submit button to enter user answers
•	Clean and engaging user interface

Social Spark Logic

The app provides the following suggestions:
•	Morning → Send a "Good morning" text to a family member
•	Mid-morning → Message a colleague with a quick "Thank you"
•	Afternoon → Share a funny meme or interesting link with a friend
•	Afternoon Snack Time → Send a "thinking of you" message
•	Dinner → Call a friend or relative for a quick catch-up
•	Night → Leave a thoughtful comment on a friend's post

 Technologies Used
 
•	Kotlin
•	Android Studio
•	Git & GitHub


Usage

1.	Open the app
2.	Enter a time of day (e.g., "Morning")
3.	View the suggested social interaction
4.	Use the reset button to try another input\
